# 기억 사항
## 작은수 % 큰수 = 작은수
```c
1 % 10 = 1
5 % 10 = 5
8 % 10 = 8
```

## if vs if-else
### if-else
-위에서부터 차례대로 검사, 처음 참이 되는 조건문만 실행하고 아래는 검사x
-여러 조건 중 하나만 실행
-학점 구하기, 메뉴 선택, 숫자가 양수 or 음수 or 0인지 판별
### if
-위의 조건이 참이어도 아래 조건도 계속 검사
-여러 조건이 모두 의미 있을 때 적합
-최댓값 갱신, 여러 조건 체크, 옵션 여러 개 동시 적용
